Software App

DDoS Penetration Testing Tool

A penetration testing tool to conduct DDoS attacks in the Application layer and Transport layer for authorized security assessments.

View Source Download for Windows

Features

Application Layer

Simulate HTTP flood and slowloris attacks at Layer 7

Transport Layer

TCP SYN flood and UDP flood testing at Layer 4

GUI Interface

Simple desktop interface for configuring and running tests

Results Analysis

View attack metrics and target response analysis

Technologies

Python Security

Quick Start

$ git clone https://github.com/Masrik-Dahir/DDoS_interface.git
$ cd DDoS_interface
$ pip install -r requirements.txt
$ python main.py

This tool is intended for authorized penetration testing and security research only. Always obtain proper authorization before testing.
