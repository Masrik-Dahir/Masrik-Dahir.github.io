Software App

Automata

Converts regex to NFA, DFA, and Minimum-DFA. Converts Context-Free Grammars to LL Grammar and Chomsky Normal Form — built for Android.

View Source Download APK

Features

Regex to NFA

Convert regular expressions to Non-deterministic Finite Automata

NFA to DFA

Transform NFA into Deterministic Finite Automata via subset construction

DFA Minimization

Minimize DFA states using partition refinement

CFG Conversion

Convert Context-Free Grammars to LL Grammar and CNF

Technologies

Java Android
