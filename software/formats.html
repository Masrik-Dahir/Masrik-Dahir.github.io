Software App

Formats

OCR, barcode, and QR code scanner with text-to-speech capabilities — built for Android.

View Source Download APK

Features

OCR Scanner

Extract text from images using optical character recognition

Barcode Scanner

Scan and decode barcodes from camera or images

QR Code Scanner

Read QR codes with instant result display

Text to Speech

Convert scanned or entered text to spoken audio

Technologies

Java Android
