Software App

Employee Management System

Node.js server with full CRUD functionalities for managing employee records, built with Fastify framework.

View Source

Features

Create

Add new employee records with full profile details

Read

Query and view employee data with filtering and pagination

Update

Modify existing employee records and profile information

Delete

Remove employee records with confirmation safeguards

Technologies

Node.js Database

Quick Start

$ git clone https://github.com/Masrik-Dahir/Fastify_server.git
$ cd Fastify_server
$ npm install
$ npm start
