Video Downloader

Download any video files from a web URL, convert video to audio, and download YouTube videos with a simple desktop interface.

View Source Download for Windows

Features

URL Download

Download video files from any web URL with automatic format detection

Video to Audio

Convert downloaded videos to audio formats like MP3

YouTube Support

Download YouTube videos directly by pasting the URL

Desktop GUI

Clean desktop interface — no command line needed

Technologies

Python

Quick Start

$ git clone https://github.com/Masrik-Dahir/Downloader.git
$ cd Downloader
$ pip install -r requirements.txt
$ python main.py

Pre-built Windows executables are available on the Releases page.
